# Discord AI

## Env

```env
OPEN_AI_API_KEY="XXXX"
BOT_TOKEN="XXXX"
SYSTEM_PROMPT="XXXX"
```

## Running

Development:

```sh
npm run dev
```

Build:

```sh
npm run build
```

Start:

```sh
npm start
```

## Setup

1. Set up the .env file according to the structure above. (use the text from system_prompt.example for the system prompt)
2. Run `npm install`.
3. Run `npm run dev`.
4. Now you can use `+chat <text>` to use the bot in chat.
